Инструкция для подписчиков HousePicker

Получение UserID

Нам необходимо узнать свой UserID в Telegram. Для этого пишем в личное сообщение боту @userinfobot в Telegram сообщение start и в ответ получаем свой id. Далее необходимо переслать полученный id саппорту HousePicker https://vk.com/housepicker

Активация подписки

Заходим в Telegram и добавляем к себе @sampgos_bot. Для активации нажимаем кнопку "Start", либо прописываем /start в сообщении.

Ура, вы подключены к HousePicker!
